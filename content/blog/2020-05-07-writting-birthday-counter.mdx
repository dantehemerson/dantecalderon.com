Primero para que nuestro sitio se vea con estilo, vamos a agregar una imagen de background.

const Container = styled.div`

Este pedazo de codigo es para que la imagen quede centrada:

```css
background-repeat: no-repeat;
background-attachment: fixed;
background-position: center;
```
