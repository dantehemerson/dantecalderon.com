I have a problem with **ESC** key in keyboards, they are incomodo difficul to reach with the the finger.  
When you look at a “correct finger placement on keyboard” image you will notice results similar to this:

![](https://substackcdn.com/image/fetch/$s_!YjUx!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F4f47f56d-c9e2-454a-ad83-e647fce5bef2_550x400.gif)

I’m not going to lie, after start using this positions to write, my WPM increase a lot.  
If you see every finger as keys “allowed” to type, and this positioning help me to improve my [WPM](https://www.typing.com/blog/what-is-words-per-minute/#:~:text=Words%20Per%20Minute%20is%20the%20number%20of%20characters%20%E2%80%94%20including%20spaces%20and%20punctuation%20%E2%80%94%20typed%20in%20one%20minute%2C%20divided%20by%20five.%20%E2%80%9CI%20can%E2%80%9D%20counts%20as%20one%20word%2C%20while%20%E2%80%9Cpopularize%E2%80%9D%20and%20%E2%80%9Clet%E2%80%99s%20type%E2%80%9D%20both%20count%20as%20two.) a lot, but something else you will notice there a keys that we daily use in that image, yep, I’m talking about the function keys and speacially the **ESC** key.

I know, when you write you don’t need to use ESC because it’s does nothing when typed in a input text. But if you a developer and use VIM, you will use it a lot. And that when this post comes in.

### Why I don’t use a small keyboard?

I know there are 60-70% keybaords that are compact and have the ESC key easy to reach, the problem with those keybaards is that they don’t come with function keys, and I find them very useful, some of the uses I do this

*   Control volume
    
*   Brigthness
    
*   Media buttons
    

### The Solution

Simple, just use your CapsLock key as ESC key. Ok maybe you are wondering, what happens with keys lock?  
If you use the template described above or if you have 50+ WPM you will problably never miss it.
